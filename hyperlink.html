Working with Hyperlinks

Below are examples of hyperlinks using the <a> tag and various target attributes:

1. Open Link in the Same Tab (Default)

Visit Wikipedia

(Opens in the same browser tab by default.)

2. Open Link in a New Tab

Search on Google

(Opens in a new browser tab or window.)

3. Open Link in a Parent Frame

Explore GitHub

(Useful when the page is loaded in an iframe. Opens in the parent frame.)

4. Open Link in the Full Window

Stack Overflow

(Opens the link in the full browser window, breaking out of any frames.)

5. Email Link

Send an Email

(Opens the default email client to send an email.)
